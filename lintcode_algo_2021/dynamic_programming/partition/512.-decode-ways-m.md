# Decode Ways 512 \(M\)

## Problem

[https://www.lintcode.com/problem/512/](https://www.lintcode.com/problem/512/)

### Description

A message containing letters from `A-Z` is being encoded to numbers using the following mapping:

```text
'A' -> 1
'B' -> 2
...
'Z' -> 26
```

Given an encoded message containing digits, determine the total number of ways to decode it.

{% hint style="info" %}
we can't decode an empty string. So you should return 0 if the message is empty. The length of message n \leq 100nâ‰¤100
{% endhint %}

### Example

**Example 1:**

```text
Input: "12"
Output: 2
Explanation: It could be decoded as AB (1 2) or L (12).
```

**Example 2:**

```text
Input: "10"
Output: 1
```

class Solution:
    """
    @param s: a string,  encoded message
    @return: an integer, the number of ways decoding
    """
    def numDecodings(self, s):
        # write your code here
        # dp[i] means the total solutions num within ith
        if not s:
            return 0
        
        n = len(s)
        dp = [0] * (n + 1)
        
        # init state
        dp[0] = 1
        dp[1] = self.decode_ok(s[0])

        for i in range(2, n + 1):
            dp[i] = dp[i - 1] * self.decode_ok(s[i - 1]) + \
                    dp[i - 2] * self.decode_ok(s[i - 2: i])
        return dp[n]
    
    def decode_ok(self, substr):
        code = int(substr)
        if len(substr) == 1 and code >= 1 and code <= 9:
            return 1
        if len(substr) == 2 and code >= 10 and code <= 26:
            return 1
        return 0
