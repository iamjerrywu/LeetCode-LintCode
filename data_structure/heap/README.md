# Heap

## Introduction

Support: 

* O\(logN\) Add 
* O\(logN\) Remove
* O\(logN\) Pop
* O\(1\) Min or Max

{% hint style="warning" %}
In Python: heapq / Java: PriorityQueue, the "remove" action require O\(n\)

* Unless it's pop the top one, otherwise, need to traverse to find the specific element
{% endhint %}

Implementation in different language

* Java: `PriorityQueue`
* C++: `priority_queue`
* Python: `heapq`
