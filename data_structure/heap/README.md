# Heap

## Introduction

Support: 

* O\(logN\) Add 
* O\(logN\) Remove
* O\(logN\) Pop
* O\(1\) Min or Max

{% hint style="warning" %}
In Python: heapq / Java: PriorityQueue, the "remove" action require O\(n\)
{% endhint %}

Implementation in different language

* Java: `PriorityQueue`
* C++: `priority_queue`
* Python: `heapq`
