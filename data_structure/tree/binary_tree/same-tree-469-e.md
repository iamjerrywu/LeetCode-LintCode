# Same Tree 469 \(E\)

## Problem



Description

Check if two binary trees are identical. Identical means the two binary trees have the same structure and every identical position has the same value.Example

**Example 1:**

```text
Input:{1,2,2,4},{1,2,2,4}
Output:true
Explanation:
        1                   1
       / \                 / \
      2   2   and         2   2
     /                   /
    4                   4

are identical.
```

**Example 2:**

```text
Input:{1,2,3,4},{1,2,3,#,4}
Output:false
Explanation:

        1                  1
       / \                / \
      2   3   and        2   3
     /                        \
    4                          4

are not identical.
```
