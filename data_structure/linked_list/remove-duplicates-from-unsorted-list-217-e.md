## Problem

Write code to remove duplicates from an _unsorted_ linked list.Example

**Example 1:**

```text
Input: 1->2->1->3->3->5->6->3->null

```

**Example 2:**

```text
Input: 2->2->2->2->2->null
Output: 2->null
```
